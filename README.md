# Reading Webiste
## Made by Michael Austoni

### Option 1 (Recommended), Start with Docker
1. **Clone The Repository**: [Git Repository](https://github.com/MichaelAA6/ReadingWebsite.git)
2. **Make Sure Dockers Installed**: [Docker](https://www.docker.com/products/docker-desktop/)
3. **Launch Container** ```docker compose up --build```
4. **Open Website** [Website]("http://127.0.0.1:5000")

### Option 2, Run Via Python

1. **Clone The Repository**: [Git Repository](https://github.com/MichaelAA6/ReadingWebsite.git)
2. **Create and Activate the Virtual Enviorment**:
   ```bash
   .\venv\Scripts\activate     # Windows
   source venv/bin/activate    # macOS/Linux
   ```
3. **Download the Requirements**:
   ```bash
   pip install -r requirements.txt

4. **Run main.py**
    ```bash
    python main.py
