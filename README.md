# Reading Stats
## Made by Michael Austoni
***
This is a website I have created to demonstrate my skills with python,flask and vega-altair.

It also follows one of my other passions which is Reading FC, and I will continue to update the page
as they play more games.

You can view my website live [HERE](https://readingstats.onrender.com/)

*(P.S Please be patient with the website as it might need time to boot and load graphs due to it being on the free tier)
***
### Option 1 (Recommended), Start with Docker
1. **Clone The Repository**: [Git Repository](https://github.com/MichaelAA6/ReadingWebsite.git)
2. **Make Sure Dockers Installed**: [Docker](https://www.docker.com/products/docker-desktop/)
3. **Launch Container** ```docker compose up --build```
4. **Open Website** [Website](http://127.0.0.1:5000)

### Option 2, Run Via Python

1. **Clone The Repository**: [Git Repository](https://github.com/MichaelAA6/ReadingWebsite.git)
2. **Create and Activate the Virtual Environment**:
   ```bash
   .\venv\Scripts\activate     # Windows
   source venv/bin/activate    # macOS/Linux
   ```
3. **Download the Requirements**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Run main.py**
    ```bash
    python main.py
    ```
